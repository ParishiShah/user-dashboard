# Dashboard Project

## Overview
This project is a dashboard built with React that displays user information, active subscriptions, and notifications. It is styled using Material UI and is designed to be responsive.

## Prerequisites
- Node.js (version 14 or later)
- npm or Yarn package manager

## Getting Started

### 1. Clone the Repository
git clone <repository-url>
cd dashboard

### 2. Install Dependencies
npm install

### 3. Start Development Server
npm start

### 4. Run Test
npm test
